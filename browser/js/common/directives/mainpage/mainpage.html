Purpose:

Provides developer teams with real-time communcation system built specifically for Git and Github.

Workflow:

Developing team would setup the local filepath to be watched. Every member then subscribe files he/she is working on for receiving real-time notification.

Functionality:

1: Tree structure view for Github repositories
2: Real-time file change notification between teams
3: Team member avaliability status
4: Group chat

Setup

Step 1: Install
Step 2: Sign in with Github account
Step 3: Select Repository
Step 4: Select local filepath for Repository
Step 5: Select files to get notifications
